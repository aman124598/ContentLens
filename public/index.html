What ContentLens does

AI-likelihood scoring

Scores text on a 1 to 10 scale using multiple language and structure heuristics.

Multiple filter modes

Blur, hide, highlight, or badge-score suspected AI-generated replies.

Fully local processing

No server calls. No raw text uploads. All scoring runs in the browser.

How it works

  1. Scan visible page content for likely reply/comment blocks.
  2. Compute local heuristic signals and combine into a single score.
  3. Apply your selected visual action based on threshold settings.

Privacy first

  • 100% local scoring
  • No identity collection
  • No browsing-history export
  • No raw text storage

ContentLens is a heuristic assistant, not a definitive AI detector.
